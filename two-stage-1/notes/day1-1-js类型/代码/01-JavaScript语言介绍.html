JavaScript历史和背景 背景：网景(火狐) 1995年，Brendan.Eich 布兰登.艾奇(克)34岁，NetScope Navigator 2 ,4月份 要求：表单验证，设计一门新的语言(不应该太复杂，需要非专业的人士也能使用，java) 实现：10天完成设计 JavaScript = C语言(基本结构) + java(语法结构和内存管理) + self(原型式继承) + scheme(函数式编程) 定义：JavaScript是一门弱类型、动态的，基于原型对象的脚本语言，支持面向对象和面向函数的编程范式。 HTML(javascript) 依赖于网页。 历史： 1995年 4月份 Brendan.Eich 入职 1995年 5月份 接到任务 - Mocha 1995年 9月份 NetScope Navigator 2 Beta2.0 liveScript 1995年 11月份 NetScope Navigator 2 Beta3.0 javaScript == sun(甲骨文) 版本情况： 说明：JavaScript - ECMAScript(ES) 标准(规范) 1996年 12月份的时候 提交JavaScript原始版本提交ECMA 标准化。 1997年 06月份的时候 ECMAScript1.0 1998年 06月份的时候 ECMAScript2.0 修改格式 1999年 12月份的时候 ECMAScript3.0 增加正则表达式，语言特性。。。 ~~~ 2009年 06月份的时候 ECMAScript5.0 ~~~ 2015年 ECMAScript6 ES2015 2016年 ECMAScript7 ES2016 2017年 ECMAScript8 2018年 ECMAScript9 ES6-ES2015版本
